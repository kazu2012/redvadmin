# REDVADMIN

![REDVADMIN][screenshot]

## Composition
- Bootstrap 4 Alpha
- Fontawasome
- jQuery
- Chartjs

## Develop
- Install Ruby and Jekyll
- Install Node, npm and grunt
- Run `jekyll serve` to generate `_site` and preview it
- Run `cd bootstrap-4` and then `grunt watch` for theming
